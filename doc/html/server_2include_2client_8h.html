Detailed Description

Déclaration des structures et fonctions pour la gestion des clients dans le jeu Awale.

Ce fichier contient les déclarations des structures et des fonctions utilisées pour gérer les clients dans le jeu Awale. Il inclut des fonctions pour initialiser, afficher, libérer et comparer les clients, ainsi que pour gérer les demandes d'amis et les informations de profil.
