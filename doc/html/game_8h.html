Detailed Description

Déclarations des fonctions et structures pour la gestion d'une partie d'Awalé.

Ce fichier contient les déclarations des fonctions et structures nécessaires pour initialiser, gérer et terminer une partie d'Awalé. Il inclut également des fonctions pour vérifier les conditions de jeu, mettre à jour le plateau, afficher le plateau, et gérer les joueurs et spectateurs.
